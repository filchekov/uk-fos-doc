Опции

Списки, чекбоксы, группы чекбоксов, радио

Каждое Значение или пара Ключ-Значение указывается с новой строки. Для указания пары Ключ-Значение, используйте знак равенства между ними. Если указано только значение, оно также будет использовано в качестве ключа.

				
					40 = Малый
					60 = Средний
					80 = Большой

Атрибуты

Для каждого поля вы можете указать HTML-атрибуты. Атрибуты должны быть указаны через пробел.

Для того, чтобы в списке по умолчанию был выбран определённый элемент, используйте атрибут data-selected="значение".

				
					readonly min="1" max="100" data-custom-attribute="custom_data"

Для появления поля только при соответствии условию, используйте data-showon="0=100", где 0 — это номер (указан слева) родительского поля, и 100 — это значение, при котором будет срабатывать условие.

Также, вместо такого значения можно использовать *, чтобы условие срабатывало когда поле заполнено и **, когда поле заполненно с учётом валидации.

Несколько условий могут быть заданы через запятую, условия для каждого родительского поля могут быть указаны через точку с запятой.

Комбинированный пример:

				
					data-showon="0=100;200;300, 1=brown, 2=mail@mail.com;**, 3=*"

Объяснение условий

Показывать поле, если одно из этих условий верно:
  1. Поле с номером 0 имеет значения 100 или 200 или 300
  2. Значение поля с номером 1 равно «brown»
  3. Поле с номером 2 равно «mail@mail.com либо заполено корректно
  4. Поле с номером 3 не пустое

Для поля типа «Файл» вы можете указать допустимые расширения файлов для загрузки, используя атрибут «accept»:

				
					accept="txt, doc, jpg, jpeg, png"
